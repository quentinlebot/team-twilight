# team-twilight

Map genrator for twilight board game

A NODEJS & MongoDB application

  #### Prerequisites
  * [nodejs](http://nodejs.org)
  * [bower](http://bower.io)
  * [mongodb](https://www.mongodb.com)

  #### Getting started :
  - npm install && bower install
  - run mongod
  - npm start
  - Visit : [http://localhost:3000](http://localhost:3000)
